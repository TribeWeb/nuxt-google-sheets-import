Any schema available for use in the Google Sheets import/export workflows needs to be defined in a module under `app/utils/googleSheetsImportSchemas.ts`.

::steps
### Define schema source

Create `app/utils/googleSheetsImportSchemas.ts`:

  :::note
  The module provides an `example` schema in `app/utils/.googleSheetsImportSchemas.ts`. Remove `.` from the start of the filename to use it.
  :::

```ts [app/utils/googleSheetsImportSchemas.ts]
import { z } from 'zod'

const example = z.object({
slug: z.string().min(1).max(100),
pageOrder: z.coerce.number().int(),
number: z.coerce.number(),
string: z.string().min(1).max(100),
enumString: z.enum(['foo', 'bar', 'baz']),
literalString: z.literal('foo'),
unionString: z.union([z.literal('foo'), z.literal('bar')]),
unionStringArray: z.union([z.literal('foo'), z.literal('bar')]).array(),
exclusiveUnionString: z.xor([z.literal('foo'), z.literal('bar')]),
stringArray: z.string().array(),
numberArray: z.coerce.number().array(),
boolean: z.coerce.boolean(),
object: z.object({
    key1: z.string().min(1),
    key2: z.string().min(1),
}),
objectArray: z.array(
    z.object({
    keyA: z.string().min(1),
    keyB: z.string().min(1),
    }),
),
})

export const googleSheetImportSchemas = {
    example,
}
```

### Map to Nuxt Content collections

Import and add the schemas to Nuxt Content collection definitions in `content.config.ts` .

```ts [content.config.ts]
import { defineContentConfig, defineCollection } from '@nuxt/content'
import { googleSheetsImportSchemas } from './utils/googleSheetsImportSchemas'

export default defineContentConfig({
    collections: {
        example: defineCollection({
            type: 'page',
            source: 'example/*.md',
            schema: googleSheetsImportSchemas.example,
        }),
    },
})

```
