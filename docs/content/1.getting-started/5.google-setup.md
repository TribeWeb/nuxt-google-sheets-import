Google Sheets Import uses API-key access. Your sheet must be publicly readable.

::warning
API-key access is appropriate for non-sensitive sheets intentionally shared as `Anyone with the link`.
For private or sensitive spreadsheets, use OAuth or service account auth (not part of this module).
::

::steps
### Set sheet permissions

1. Open your sheet in Google Sheets.
2. Click `Share`.
3. Set `General access` to `Anyone with the link`.
4. Set role to `Viewer`.
5. Copy the spreadsheet ID from:

```text
https://docs.google.com/spreadsheets/d/<SPREADSHEET_ID>/edit
```

If your Workspace policy prevents public-link sharing, API-key access will fail.

### Create a Google API key

1. Open Google Cloud Console.
2. Create or select a project.
3. Enable Google Sheets API.
4. Create credentials -> API key.
5. Restrict the key:
   - API restrictions: `Google Sheets API`
   - Application restrictions:
     - server: IP addresses (recommended)
     - browser-only: HTTP referrers
6. Add the key to your local environment variables:

```bash [.env]
NUXT_GOOGLE_API_KEY=your_google_sheets_api_key
```

### Configure runtime key mapping

Map the module option to your runtime config key:

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  runtimeConfig: {
    googleApiKey: '',
  },
})
```
