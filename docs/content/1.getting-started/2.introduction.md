Welcome to **Google Sheets Import**, a Nuxt module for seamless integration between your Nuxt Content frontmatter and Google Sheets.

## Why Google Sheets Import?

There are several ways to manage and edit frontmatter in a Nuxt Content project, but they often involve individual manual file edits, which can be time-consuming and error-prone. The initial setting up of multiple pages or data records can be particularly tedious, especially when creating catalogs, inventories, or any structured content that benefits from a tabular format.
In-app bulk editing is relatively straightforward but it nevertheless requires custom code which must be maintained. Google Sheets Import offers a unique approach by leveraging the familiarity and collaborative features of Google Sheets to manage your content's frontmatter data.

Nuxt Content already requires that each page type or data set is defined using Zod schemas, which provide a clear structure for content frontmatter. Google Sheets Import builds on this by allowing you to map your Zod schemas to Google Sheets columns, enabling you to edit your content in a spreadsheet format. This not only simplifies the bulk editing process but also allows for collaboration with team members who may not be comfortable editing markdown files directly.
