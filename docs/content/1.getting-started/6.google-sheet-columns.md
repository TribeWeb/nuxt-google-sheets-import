---
title: Google Sheet column headers
description: Set up column headers in the Google Sheet so they map to the schema.
navigation:
  icon: i-teenyicons-spreadsheet-outline
---

Google Sheets must be set up so that the column headers correspond to the properties defined in your Zod schema.

::steps
### Go to the import page

With your application in `dev`, navigate to `/google-sheets-import`. Select the `Setup Google Sheet` tab.

### Select a schema

From the dropdown, select the schema you would like to work with.

  :::note
  If you do not see your schema listed, ensure it is [configured correctly](/getting-started/schema-setup).
  :::

### Copy and paste into your Google Sheet

Copying as a tab seperated row allows you to paste the column headings directly into a blank Google Sheet.

  :::note
  To prepare Nuxt Content page metadata overrides like `title` ready for import, add appropriate column headers from the list provided.
  :::

  :::warning
  Nuxt Content page metadata overrides don't need to be added to the schema.
  :::
::
