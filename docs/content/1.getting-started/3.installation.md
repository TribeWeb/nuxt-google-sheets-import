## Install Nuxt Google Sheets Import

::steps
### Install the module

In your [Nuxt Content](https://content.nuxt.com) project, install the module:

  :::code-group
  ```bash [pnpm]
  pnpm add -D nuxt-google-sheets-import
  ```

  ```bash [yarn]
  yarn add -D nuxt-google-sheets-import
  ```

  ```bash [npm]
  npm install -D nuxt-google-sheets-import
  ```

  ```bash [bun]
  bun add -D nuxt-google-sheets-import
  ```
  :::

  :::note
  Ensure that [`Zod`](https://zod.dev/) is also installed as a declared dependency. See [using Zod v3](https://content.nuxt.com/docs/collections/validators#using-zod-v3) and [using Zod v4](https://content.nuxt.com/docs/collections/validators#using-zod-v4) in the Nuxt Content documentation.
  :::

### Add the module to your Nuxt config

```ts [nuxt.config.ts]
export default defineNuxtConfig({
modules: ['nuxt-google-sheets-import']
})
```

### Configure the module

```ts [nuxt.config.ts]
export default defineNuxtConfig({
modules: ['nuxt-google-sheets-import'],
googleSheetsImport: {
  apiBase: '/api/google-sheets-import', // custom API base path
  defaultContentDir: 'content', // directory for imported content
  googleSheets: [
    { id: '<SPREADSHEET_ID_1>', label: 'Primary Sheet' },
    { id: '<SPREADSHEET_ID_2>', label: 'Staging Sheet' },
  ],
},
})
```

- `apiBase`: base path for module API routes.
- `defaultContentDir`: default write target when saving imported records.
- `googleSheets`: spreadsheet options shown in the `Import data` source dropdown.

  :::note
  Google Sheets Import runs in `dev` mode only.
  :::
::
