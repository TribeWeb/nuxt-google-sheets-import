## Install Nuxt Google Sheets Import

::steps
### Install the module

In your Nuxt project, install the module:

  :::code-group
  ```bash [pnpm]
  pnpm add -D nuxt-google-sheets-import
  ```

  ```bash [yarn]
  yarn add -D nuxt-google-sheets-import
  ```

  ```bash [npm]
  npm install -D nuxt-google-sheets-import
  ```

  ```bash [bun]
  bun add -D nuxt-google-sheets-import
  ```
  :::

### Add the module to your Nuxt config

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  modules: ['nuxt-google-sheets-import']
})
```

### Configure the module

```ts [nuxt.config.ts]
export default defineNuxtConfig({
modules: ['nuxt-google-sheets-import'],
googleSheetsImport: {
apiBase: '/api/google-sheets-import', 
// optional custom API base path
googleApiKeyRuntimeKey: 'googleApiKey', 
// runtime key for Google API
defaultContentDir: 'content/data' 
// default directory for imported content
}
})
```

  :::note
  Google Sheets Import runs in `dev` mode only.
  :::
::
