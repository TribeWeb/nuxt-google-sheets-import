Google Sheets must be set up so that the column headers correspond to the properties defined in your Zod schema.

::steps
### Go to the module page

With your application running in `development mode`, navigate to `/google-sheets-import`. Select the `Setup Google Sheet` tab.

### Select a schema

From the dropdown, select the schema you would like to work with.

  :::warning
  If you do not see your schema listed, ensure it is [configured correctly](/getting-started/schema-setup).
  :::

### Copy and paste into your Google Sheet

Copying as a tab seperated row allows you to paste the column headings directly into a blank Google Sheet.

  :::note
  To prepare [Nuxt Content page schema overrides](https://content.nuxt.com/docs/collections/types#schema-overrides) like `title` or `seo` ready for importing, add appropriate column headers from the list provided to the Google Sheet.
  :::

  :::warning
  Nuxt Content page schema overrides don't need to be added to your schema definition.
  :::
::
